	\section{ÉCRIRE LE TEXTE SUIVANT.}
	\noindent Une fonction $ f : [0, 1] \to \mathbb{R} $ est continue si
	\[
		\forall x \in [0, 1], \forall \varepsilon > 0, \exists \delta > 0, \forall y \in [0, 1] : |y-x| < \delta \Rightarrow |f(y)-f(x)|<\varepsilon 
	\]
